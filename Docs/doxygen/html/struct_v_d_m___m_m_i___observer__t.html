Detailed Description

The MMI Observer monitors the MMI component. MMI APIs must notify the MMI Observer whenever a user responds to a query. The MMI Observer then notifies the Engine.

The MMI Observer is a structure of callback pointers passed to the MMI component upon initialization and made available to all MMI APIs.
