typedef void(* VDM_DL_Ext_onAbortCB_t)(IBOOL inUserInitiated)

Notification invoked by the Engine after the user cancels an Alternative Download. If inUserInitiated is TRUE, implementation should cancel the session. If FALSE, the session should be aborted.

Parameters
inUserInitiatedTRUE if the user cancelled the download, FALSE otherwise
Returns
VDM_ERR_OK on success, or a Errors error code
