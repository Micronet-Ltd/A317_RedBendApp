Detailed Description

Persistent Storage APIs.

OMA DM Protocol Engine does not assume that the device supports a file system. The storage APIs support opening and closing a storage item, reading and writing to and from an open storage item, and deleting a storage item from persistent storage.

In operating systems that use a file system, a storage item is typically a file.
