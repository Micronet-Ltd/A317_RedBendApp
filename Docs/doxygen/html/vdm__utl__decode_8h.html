Encode data using Base64.

The returned buffer has a NULL-terminator. This is useful if it is necessary to turn the data into a string. The NULL-terminator is NOT included in the returned length.

Parameters
inDecDataThe data to encode.
inDecLengthLength of inDecData (in bytes).
outEncDataA buffer to store the encoded data, or NULL on error; the caller must free this buffer.
outEncLengthLength of data in outEncData, excluding NULL-terminator.
Returns
VDM_ERR_OK on success or a VDM_ERR_defs error code
