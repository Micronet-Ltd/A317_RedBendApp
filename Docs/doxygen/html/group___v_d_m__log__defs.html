Detailed Description

The following macros provide a more user-friendly API for logging. The format of "inMsg" is the standard printf arguments: (const char * format, ...)

USAGE: The format string and arguments must be enclosed with double parentheses. For example: 

VDM_log(("format", args...));

Dump formatted lines of 16 hex bytes, preceded by a line of text. For example: 

Content of buffer is:
0000  48 54 54 50 20 70 6F 72 74 3D 38 32 33 36 20 68  HTTP port=8236 h
0010  6F 73 74 3D 2A                                   ost=*
Note
This only dumps to the Debug or Verbose log levels.

Usage: VDM_logDebugFormattedHex2(msg, buf, len); VDM_logVerboseFormattedHex2(msg, buf, len);
