VDM_Error VDM_UTL_DynBuffer_reset ( VDM_UTL_DynBuffer_t *  inBuffer)

Reset dynamic buffer to its initial size and zero out buffer (reset all data to \0).

Note
Resetting the buffer to its original size frees and reallocates memory, so there is a slight chance of failure on low memory.
Parameters
inBufferPointer to the dynamic buffer, a VDM_UTL_DynBuffer_t structure
Returns
VDM_ERR_OK on success or VDM_ERR_MEMORY if there is no memory to perform the action.

Set data at a specific offset in a dynamic buffer.

If inOffset + inDataLen exceeds current buffer length, the buffer is resized as required, up to the maximum set in VDM_UTL_DynBuffer_create.

Note
Existing data at the offset is overwritten!
