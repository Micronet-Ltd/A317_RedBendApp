Detailed Description

Asynchronous Priority Queue API.

The message queue utility is based on the priority queue vdm_utl_priqueue.h. Added functionality includes:

  • An asynchronous object is signalled whenever an item is inserted to the queue.
  • Access to the queue is protected with a mutex.

Lower priority means that an item is more important.

typedef IBOOL(* VDM_UTL_MsgQueue_conditionalDestroyCB)(void *inPayload, void *inParam)

Callback invoked to free memory of queued item from a message queue according to an external condition.

The callback is invoked by VDM_UTL_MsgQueue_condRemove. When the condition is met, the callback must free the memory allocated for the item. If this callback returns TRUE, then the item has been removed from the queue.

Parameters
inPayloadThe payload to free.
inParamAdditional parameters passed to the callback.
Returns
TRUE if condition was met and the item was freed, FALSE otherwise
