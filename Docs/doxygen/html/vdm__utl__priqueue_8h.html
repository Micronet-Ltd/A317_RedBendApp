Detailed Description

Priority Queue API.

Lower priority means that an item is more important; 0 is the highest priority.

typedef IBOOL(* VDM_UTL_PriQueue_conditionalDestroyCB)(void *inPayload, void *inParam)

Callback invoked to destroy a queued item in a priority queue if it matches a condition. The function should free the memory of the destroyed item.

Invoked by VDM_UTL_PriQueue_condRemove.

Parameters
inPayloadA pointer to the item
inParamA pointer to any additional parameters
Returns
TRUE if the condition was met and the item was destroyed, FALSE otherwise
