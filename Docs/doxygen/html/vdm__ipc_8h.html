Detailed Description

Inter Process Communication.

The SMM Engine runs first and waits for the DIL to connect using IPC. SMM Engine and DIL are peers, client and server to each other.

Android doesn't use IPC functions, but uses the package and parse functions (VDM_IPC_genEvent and VDM_IPC_parseEvent).
