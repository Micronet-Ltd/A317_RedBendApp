Detailed Description

ECU IPL Transmitter APIs.

Enumeration Type Documentation

enum E_SWM_PROT_DataType_t

Values for the structure union. For an operation like install or update, the request includes the DP to install or update as a data in this union. The response is typically a string or number in this union.

enum E_SWM_PROT_InsPhase_t

Installation phase – Update of ECU can be done in few phases which are informed with this enum

enum E_SWM_PROT_Tag_t

The action to perform on the ECU or the response that is expected
